# Client

Клиентская часть текущей версии представляет собой интерфейс на чистом HTML, CSS и JavaScript.

Сейчас здесь реализованы:

- форма настройки дневных целей;
- форма добавления приёма пищи;
- список записей за день;
- карточки прогресса по КБЖУ.

Позже этот каталог можно заменить на полноценное SPA-приложение на выбранном фреймворке.
