# Client

Клиентская часть проекта выполнена на `HTML + CSS + JavaScript`.

## Что Уже Реализовано

- авторизация и регистрация;
- рабочая панель пользователя с дневной аналитикой;
- цели КБЖУ и формы быстрого ввода;
- журнал приёмов пищи и фильтрация по типам;
- каталог продуктов и административный режим;
- шаблоны, рецепты, планирование и избранное;
- вода, самочувствие, заметки дня и замеры тела;
- экспорт в `JSON` и `PDF`;
- светлая и тёмная темы;
- адаптивная навигация по разделам.

## Ключевые Файлы

- [index.html](./index.html) — структура интерфейса;
- [styles.css](./styles.css) — визуальная система и адаптивность;
- [app.js](./app.js) — клиентская логика, API-запросы и обновление состояния.

## Назначение

Текущий интерфейс ориентирован на читаемую, не перегруженную рабочую панель пользователя, чтобы проект выглядел как полноценная прикладная система, а не как учебная заготовка.
