# Персональный дневник питания с анализом КБЖУ

Клиент-серверное приложение для ведения дневника питания и контроля дневных целей по калориям, белкам, жирам и углеводам. В репозитории уже есть рабочая стартовая версия с веб-интерфейсом, серверным API и автоматическим расчётом дневной сводки.

## Что уже реализовано

- хранение дневных целей по калориям, белкам, жирам и углеводам;
- отображение прогресса по целям за день;
- добавление приёмов пищи через форму;
- удаление записей из дневника;
- расчёт суммарных КБЖУ по текущим записям;
- запуск без базы данных и внешних пакетов, только на встроенных возможностях Node.js.

## Быстрый запуск

Требование: Node.js 22 или совместимая версия.

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8080
```

## API приложения

- `GET /api/health` - статус сервера.
- `GET /api/goals` - текущие дневные цели.
- `PUT /api/goals` - обновить цели.
- `GET /api/meals` - список приёмов пищи.
- `POST /api/meals` - добавить запись.
- `DELETE /api/meals/{id}` - удалить запись.
- `GET /api/summary` - сводка по КБЖУ за день.

## Текущие ограничения

- данные пока хранятся только в памяти сервера;
- авторизация и аутентификация ещё не реализованы;
- база данных пока не подключена;
- frontend и backend пока развиваются в одной минимальной сборке.
