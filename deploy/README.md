# Deploy

Каталог зарезервирован под контейнеризацию и развёртывание.

На текущем этапе серверный Dockerfile уже умеет запускать полноценное приложение c SQLite-базой, Swagger и клиентской частью. Клиентский Dockerfile оставлен как отдельная заготовка под будущее разделение frontend и backend.

После выбора окончательного стека сюда можно будет добавить:

- production и development Dockerfile;
- compose-файлы для локального запуска;
- шаблоны конфигурации окружения;
- инструкции по деплою в облако;
- продвинутые варианты CI/CD-конфигурации.
