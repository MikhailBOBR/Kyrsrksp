# Server

Серверная часть проекта реализована на `Node.js + Express + SQLite`.

## Что Находится В Сервере

- модульная структура `routes + service`;
- JWT-аутентификация и ролевая модель `user/admin`;
- работа с базой данных, инициализация схемы и seed-данные;
- дневник питания, каталог продуктов, цели КБЖУ;
- вода, самочувствие, замеры тела, заметки дня;
- шаблоны, рецепты, планировщик, покупки, избранное;
- Swagger UI и OpenAPI JSON;
- набор автоматических тестов на `node:test`.

## Ключевые Каталоги

- [src/modules](./src/modules) — доменные модули приложения;
- [src/middlewares](./src/middlewares) — auth, RBAC, обработка ошибок;
- [src/db](./src/db) — схема, подключение и сидирование базы;
- [tests](./tests) — API, smoke, contract, fuzz и static tests.

## Следующий Технический Шаг

Если проект потребуется масштабировать дальше, серверный слой можно перенести с `SQLite` на `PostgreSQL` без изменения общей модульной структуры.
