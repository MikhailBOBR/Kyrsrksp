# Server

Серверная часть текущей версии написана на Node.js без внешних зависимостей.

Что уже есть:

- HTTP API для целей пользователя и списка приёмов пищи;
- расчёт дневной сводки по калориям, белкам, жирам и углеводам;
- валидация входных данных;
- раздача статических файлов клиентского интерфейса.

Следующий шаг для сервера - перейти от in-memory хранения к базе данных и добавить аутентификацию.
